# Проект интернет-магазина
## Описание

В проекте используются Django, Bootstrap

1. Клонируйте репозиторий:
```
git clone https://github.com/username/homework22.git
```

## Использование:

1. Откройте проект в вашей среде разработки.
2. Запустите проект
```
python3 manage.py runserver
```

## Модуль `config`:
Используется для конфигурации проекта.

## Модуль `Catalog`:
Сожержит файлы и директории приложения, такие как контроллеры, html-страницы.

## `manage.py`:
Скрипт командной строки, созданный Django.
